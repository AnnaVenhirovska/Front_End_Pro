1. Создаем 2 блока, у каждого блока есть кнопка Click и счетчик counter(в виде числа).
2. При нажатии на Click - counter увеличивается. При перезагрузке страницы counter должен сохраняться.
3. Создать кнопку ClearCounters()
4. Создать кнопку setCounter(), который запрашивает id блока и устанавливает значение( в типе number ) в counter.
5. При увеличении counter-а: каждое значение до 50 должно окрашивать соответствующий блок в случайно сгенерированный цвет. Цвета генерируются при изменении counter-a и сохраняются в Storage. Когда значение доходит до 50 - применяется цвет по умолчанию. 'this.style.backgroundColor = "rgb(x, y, z)";' -> x = getRand(256) y = getRand(256) z = getRand(256)
