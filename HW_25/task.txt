Создать react SPA приложение.

Страницы:

- Sign In/Sign Up (авторизация и создания аккаунта)
- Home
- Contact us


1) На странице Sign In/Sign Up пользователь может **создать** аккаунт. Для этого используется форма с введением логина, пароля и повторение пароля, что должны отправится на сервер по ендпоинту `/auth/create`.

2) На странице Sign In/Sign Up пользователь может **авторизироватся**. Для этого используется форма с введением логина и пароля, что должны отправится на сервер по ендпоинту `/auth/login`

3) Страница Home - под защитой от неавторизированных пользователей. На странице Home отображаются все пользователи зарегестрированые на нашем сайте. Отображение должно быть в виде таблицы с такими даными - login, creation time, last login time. Получить список нужно по запросу на ендпоинт `/users/all`

4) Страница Contact us - под защитой от неавторизированных пользователей. На странице отображается форма обратной связи где можно написать любой текст в textarea. Вопрос будет отправлен на сервер по endpoint `/contact/ask`


Сервер:

1) Создать сервер по аналогии с лекцией #40

2) Хранить всех созданный пользователей в json файле - `users-in-system.json`

3) Хранить все сообщения от формы Contact us в json файле - `contact-questions.json`

4) Каждый login пользователя в существующий аккаунт должен ОБНОВИТЬ поле `lastLoginTime`

5) Создание аккаунта должно добавлять нового юзера в `users-in-system.json`, с полями - loginName, password (in hash), creationTimem, lastLoginTime
